# Macro Tracker

This is app is a macro diary.  Try the live demo [here!](https://macro-tracker-do917.herokuapp.com/)

## To Install
```
git clone https://github.com/do917/macro-tracker.git
cd macro-tracker/
yarn install
yarn start
```
